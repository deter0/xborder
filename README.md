## Dependencies
* xdotool (On Arch it's `pacman -S xdotool`)
* python 3.10 (3.9 doesn't work!)
* pycairo (Tested with version 1.21.0, `pip install pycairo`)
* gtk

### Configuration
Configuration can be edited in the "config.json" file.
